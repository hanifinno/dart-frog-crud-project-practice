# Some commands for dart_frog
- `dart_frog create <project_name>`: Create a new Dart Frog project.
- `dart_frog dev`: Start the development server.
- `dart_frog build`: Build the Dart Frog application for production.
- `dart_frog test`: Run tests for the Dart Frog application.
- `dart_frog analyze`: Analyze the Dart Frog application for potential issues.
